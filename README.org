* Requirements for Development
- Node
- web-ext
- GNU Make
* Development
** Prepare
#+BEGIN_SRC sh
npm i
#+END_SRC
** Run for development (Watch)
#+BEGIN_SRC sh
make dev
web-ext run --verbose --firefox-profile ~/.mozilla/firefox/PROFILE_DIR_NAME
#+END_SRC
** Deploy (Build .xpi file)
#+BEGIN_SRC sh
make xpi
#+END_SRC
